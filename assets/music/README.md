
# Folder Musik ValMusic

Simpan file musik Anda di sini agar otomatis terdeteksi oleh aplikasi.
Format penamaan file harus sama persis untuk sinkronisasi lirik.

Contoh:
- `assets/music/lagu_galau.flac` (File Audio)
- `assets/music/lagu_galau.lrc` (File Lirik)

Supported Audio: mp3, flac, wav, ogg, m4a
Supported Lyrics: lrc
